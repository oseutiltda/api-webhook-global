# Imagem de Capa do Login

## Instruções

Coloque a imagem de capa do login nesta pasta (`frontend/public/`) com o nome:

**`capa1.png`**

A imagem deve ser uma ilustração de logística global, transporte e entrega, conforme descrito no design do login.

## Estrutura de Arquivos

```
frontend/
  ├── public/
  │   └── capa1.png  ← Coloque a imagem aqui
  └── src/
      └── components/
          └── LoginForm.tsx  ← Componente que usa a imagem
```

## Nota

O componente `LoginForm.tsx` está localizado em `frontend/src/components/LoginForm.tsx` e já está configurado para usar a imagem `/capa1.png` da pasta public.
